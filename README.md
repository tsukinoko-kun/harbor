# Harbor

<img src="https://raw.githubusercontent.com/tsukinoko-kun/harbor/refs/heads/main/icon.png" width="64">

Docker GUI

## Install

### macOS

```sh
brew install --cask tsukinoko-kun/tap/harbor
```

### Linux

```sh
brew install tsukinoko-kun/tap/harbor
```

Since Linux does not support Casks, you can only run Harbor using its command.
